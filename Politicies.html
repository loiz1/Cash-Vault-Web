Política de Privacidad

Última actualización: 13 de marzo de 2026

Introducción

CashVault es una aplicación de finanzas personales diseñada para ayudar a los usuarios a registrar y analizar sus ingresos, gastos, presupuestos y metas de ahorro. Esta Política de Privacidad describe cómo se recopila, utiliza, almacena y protege la información cuando utilizas la aplicación.

Al usar CashVault aceptas las prácticas descritas en esta política.

Información que recopilamos

Información proporcionada por el usuario

Cuando utilizas la aplicación podemos recopilar información que ingresas directamente, incluyendo:

  • Dirección de correo electrónico utilizada para crear la cuenta.
  • Datos financieros que registras en la app (ingresos, gastos, categorías, presupuestos, metas de ahorro, notas de transacciones).
  • Información relacionada con miembros del hogar si decides utilizar la función de finanzas compartidas.

Esta información se utiliza únicamente para el funcionamiento de la aplicación.

Información generada por el uso de la aplicación

Podemos recopilar información técnica básica para mejorar el funcionamiento del servicio, como:

  • Identificadores del dispositivo
  • Registros de uso de la aplicación
  • Información de rendimiento y errores

Esta información se utiliza para mejorar la estabilidad, seguridad y rendimiento de la app.

Cómo usamos tu información

La información recopilada se utiliza exclusivamente para:

  • Crear y administrar tu cuenta de usuario
  • Guardar y sincronizar tus datos financieros
  • Mostrar estadísticas, reportes y gráficos dentro de la aplicación
  • Mejorar la experiencia del usuario
  • Mantener la seguridad del servicio
  • Enviar recordatorios o notificaciones configuradas por el usuario
Nota importante: CashVault no vende ni comparte tus datos financieros con terceros con fines comerciales.

Almacenamiento de datos

Los datos se almacenan utilizando servicios de infraestructura de Google Firebase, que pueden incluir:

  • Firebase Authentication para gestionar cuentas de usuario
  • Cloud Firestore o Realtime Database para almacenar datos
  • Firebase Cloud Messaging para notificaciones

Estos servicios aplican medidas de seguridad y cifrado estándar de la industria.

Seguridad de la información

Se implementan medidas técnicas y organizativas para proteger la información del usuario contra accesos no autorizados, pérdida o alteración.

Esto incluye:

  • Autenticación segura mediante Firebase
  • Conexiones cifradas (HTTPS)
  • Control de acceso a los datos en la nube
Nota de seguridad: Sin embargo, ningún sistema puede garantizar seguridad absoluta.

Compartición de datos

CashVault no vende ni alquila datos personales.

La información solo puede compartirse en los siguientes casos:

  • Cuando sea requerido por ley o autoridades legales
  • Con proveedores de infraestructura necesarios para operar la app (por ejemplo Google Firebase)

Control de tus datos

El usuario mantiene el control total sobre sus datos financieros.

Puedes:

  • Editar o eliminar transacciones dentro de la app
  • Eliminar tu cuenta y datos asociados
  • Dejar de usar la aplicación en cualquier momento
Nota importante: Cuando eliminas tu cuenta, se eliminarán también los datos asociados almacenados en nuestros servicios.

Privacidad de menores

La aplicación no está dirigida a menores de 13 años. No recopilamos intencionalmente información personal de menores.

Si detectamos que se ha recopilado información de un menor sin consentimiento parental, tomaremos medidas para eliminarla.

Cambios en esta política

Esta Política de Privacidad puede actualizarse ocasionalmente para reflejar cambios en la aplicación o en requisitos legales.

Las actualizaciones se publicarán dentro de la aplicación o en la página oficial del proyecto.

Contacto

Si tienes preguntas sobre esta Política de Privacidad puedes contactarnos a través del correo electrónico de soporte de la aplicación.
